# 🎨 流程图模块 - 立即使用

## ✅ 模块已完成！

流程图编辑模块已完全实现，包含所有核心功能。

---

## 🚀 三步开始使用

### 第一步：安装依赖（必须）
```bash
cd frontend
pnpm install
```

### 第二步：启动服务
```bash
# 在项目根目录
./start.sh
```

### 第三步：创建流程图
1. 登录教师账号（teacher@inspireed.com / teacher123）
2. 打开或创建教案
3. 点击 **"📊 流程图单元"**
4. 开始创作！

## 💡 快速示例

创建一个简单流程：

```
开始 → 过程1 → 判断
              ├─ 是 → 过程2 → 结束
              └─ 否 → 过程3 → 结束
```

1. 添加节点（点击工具栏按钮）
2. 连接节点（拖拽连接点）
3. 自动布局（点击🎯按钮）
4. 保存教案

## ⚡ 常见问题

**Q: 安装依赖失败？**
```bash
cd frontend
rm -rf node_modules pnpm-lock.yaml
pnpm install
```

**Q: 流程图不显示？**
- 检查依赖是否安装
- 刷新浏览器页面
- 查看控制台错误

**Q: 如何导出图片？**
- 点击工具栏的 📷 按钮
