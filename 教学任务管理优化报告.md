## ✅ 优化方案

### 1. 自动关联教师所属学校

**实现逻辑**：
```typescript
// 当选择教师时
function handleTeacherChange(teacherId: number) {
  const teacher = teachers.value.find(t => t.id === teacherId)
  if (teacher && teacher.school_id) {
    // ✅ 自动填充学校ID
    formData.value.school_id = teacher.school_id
    isSchoolAutoFilled.value = true

    // 清空年级和班级（因为学校变了）
    formData.value.grade_id = 0
    formData.value.classroom_id = 0
  }
}
```

**用户体验改进**：
1. 选择教师后，学校字段**自动填充**
2. 学校选择器显示提示："📍 根据所选教师自动填充"
3. 教师选择器显示教师所属学校："张三 (开平市第一中学)"
4. 用户仍可手动修改学校（如果需要跨校教学）

**智能标记**：
- 如果用户手动修改了学校，自动取消"自动填充"标记
- 编辑时自动检测学校是否来自教师所属学校

### 2. 自动填充学年

**实现逻辑**：
```typescript
function getCurrentAcademicYear(): string {
  const now = new Date()
  const currentYear = now.getFullYear()
  const month = now.getMonth() + 1

  // 如果是8月及以后，属于下一学年
  if (month >= 8) {
    return `${currentYear}-${currentYear + 1}`  // 如：2025-2026
  } else {
    return `${currentYear - 1}-${currentYear}`  // 如：2024-2025
  }
}
```

**规则**：
- 8月及以后 → 下一学年
- 1-7月 → 当前学年

**示例**：
- 2025年1月 → `2024-2025`
- 2025年9月 → `2025-2026`

---

### 3. 自动填充学期

**实现逻辑**：
```typescript
function getCurrentSemester(): Semester | undefined {
  const now = new Date()
  const month = now.getMonth() + 1

  // 第一学期 9-1月，第二学期 2-7月
  const isSecondSemester = month >= 2 && month <= 7

  return semesters.value.find(semester => {
    const semesterName = semester.name.toLowerCase()
    if (isSecondSemester) {
      return semesterName.includes('二') ||
             semesterName.includes('下') ||
             semesterName.includes('spring')
    } else {
      return semesterName.includes('一') ||
             semesterName.includes('上') ||
             semesterName.includes('fall')
    }
  })
}
```

**规则**：
- 9月-次年1月 → 第一学期（上学期/Fall）
- 2月-7月 → 第二学期（下学期/Spring）
