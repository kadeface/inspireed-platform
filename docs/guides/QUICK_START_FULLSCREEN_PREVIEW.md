# 🎬 全屏预览功能快速开始指南

> 2 分钟快速体验教师端全屏预览功能

---

## 第一步：启动应用

```bash
# 在项目根目录执行
./start.sh
```

等待服务启动完成，你会看到：
- 前端服务：http://localhost:5173
- 后端服务：http://localhost:8000

1. 打开浏览器访问：http://localhost:5173

2. 使用教师账号登录：
   - 邮箱：`teacher@inspireed.com`
   - 密码：`teacher123`

---

## 第三步：创建或打开教案

### 方式 A：创建新教案（推荐用于测试）

1. 在教师工作台页面，点击"创建教案"按钮

2. 添加几个不同类型的单元：
   - 点击左侧工具栏的"文本"按钮，添加一个文本单元
   - 点击"代码"按钮，添加一个代码单元
   - 点击"视频"按钮，添加一个视频单元

3. 随意编辑内容（用于测试预览效果）

### 方式 B：打开已有教案

1. 在教师工作台页面，找到任意已有的教案

2. 点击教案卡片进入编辑页面

---

## 第四步：体验全屏预览 ⭐

### 进入全屏预览

在教案编辑页面顶部工具栏右侧，找到 **紫色的"全屏预览"按钮**（带眼睛图标），点击它。

```
┌──────────────────────────────────────────────────────┐
│ ←  教案标题    [保存] [发布] [预览模式] [全屏预览]  │ ← 点击这里
└──────────────────────────────────────────────────────┘
```

### 体验全屏效果

进入全屏预览后，你会看到：
- ✅ 页面全屏显示教学内容
- ✅ 编辑工具栏已隐藏
- ✅ 顶部只保留简洁的导航栏
- ✅ 右下角有"返回顶部"浮动按钮

### 测试滚动功能

如果教案内容较多：
1. 向下滚动到页面底部
2. 点击右下角的圆形"返回顶部"按钮
3. 观察平滑滚动效果

### 退出全屏预览

有两种方式：
1. **点击按钮**：点击右上角的"退出预览"按钮
2. **键盘快捷键**：按下 `Esc` 键（推荐！）

## 💡 小贴士

1. **流畅的动画**  
   注意观察进入和退出全屏时的平滑过渡动画

2. **键盘快捷键**  
   使用 `Esc` 键退出最快捷

3. **自定义滚动条**  
   全屏预览模式下的滚动条经过优化，视觉效果更好

4. **内容居中**  
   内容区域最大宽度约 1280px，自动居中显示

5. **背景滚动锁定**  
   进入全屏预览后，页面背景无法滚动，避免干扰

## 遇到问题？

### 问题 1：点击"全屏预览"按钮没反应
**解决方案：**
- 检查浏览器控制台是否有错误信息
- 刷新页面重试
- 确认使用的是支持的浏览器（Chrome、Firefox、Safari）

### 问题 2：按 Esc 键无法退出全屏预览
**解决方案：**
- 确认焦点在页面上（点击页面任意位置）
- 使用"退出预览"按钮替代

### 问题 3：动画效果卡顿
**解决方案：**
- 检查浏览器性能
- 关闭不必要的浏览器扩展
- 尝试更新浏览器到最新版本
