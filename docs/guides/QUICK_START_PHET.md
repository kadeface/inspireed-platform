# 🚀 PhET 风格设计 - 快速开始指南

## 📋 概述

本指南将帮助您快速了解和使用 InspireEd 平台的 PhET 风格设计改进。

## ✨ 新功能亮点

### 1. 🎨 全新的浏览页面
- 路径：`/student/browse`
- 类似 PhET 的色彩丰富设计
- 学科分类卡片（物理、数学、化学、生物、地球科学）
- PhET 互动模拟实验集成

### 2. 🔬 PhET 模拟实验
- 30+ 官方 PhET 模拟实验
- 直接链接到 PhET.colorado.edu
- 涵盖多个学科领域

### 3. 🎯 改进的学生仪表板
- 醒目的 PhET 探索横幅
- 一键访问浏览页面

## 🎬 如何使用

### 步骤 1: 启动项目

```bash
# 进入前端目录
cd frontend

# 安装依赖（如果还没安装）
npm install

# 启动开发服务器
npm run dev
```

### 步骤 2: 登录系统

1. 访问 `http://localhost:5173/login`
2. 使用学生账号登录
3. 自动跳转到学生仪表板

### 步骤 3: 探索新功能

#### 方式 1: 通过仪表板
1. 在学生仪表板顶部看到 PhET 探索横幅
2. 点击 **"立即探索 →"** 按钮
3. 进入浏览页面

#### 方式 2: 直接访问
- 直接访问 `http://localhost:5173/student/browse`

### 步骤 4: 使用浏览功能

#### 搜索课程
- 在顶部搜索栏输入关键词
- 实时筛选课程和模拟实验

#### 按学科筛选
- 点击任意学科卡片（物理、数学、化学等）
- 显示该学科的课程和模拟实验
- 再次点击取消筛选

#### 体验 PhET 模拟实验
1. 向下滚动到 "PhET 互动模拟实验" 区域
2. 点击任意模拟实验卡片
3. 在新窗口打开 PhET 官方模拟实验

#### 收藏课程
- 点击课程卡片右上角的心形图标
- 红色表示已收藏，灰色表示未收藏

#### 排序课程
- 使用右上角的排序下拉菜单
- 选项：默认排序、评分最高、最受欢迎、最新发布

## 🔧 配置说明

### PhET 模拟实验数据

位置：`frontend/src/data/phet-simulations.ts`

添加新的模拟实验：

```typescript
{
  id: 'your-simulation-id',
  name: 'English Name',
  nameCn: '中文名称',
  category: 'physics', // physics, math, chemistry, biology, earth
  description: 'English description',
  descriptionCn: '中文描述',
  url: 'https://phet.colorado.edu/sims/...',
  topics: ['topic1', 'topic2']
}
```

### 路由配置

位置：`frontend/src/router/index.ts`

新路由已添加：

```typescript
{
  path: '/student/browse',
  name: 'StudentBrowse',
  component: () => import('../pages/Student/BrowseSimulations.vue'),
  meta: { requiresAuth: true, role: 'student' }
}
```

## 🐛 常见问题

### Q1: 无法访问浏览页面？
**A:** 确保您已登录学生账号，该页面需要学生权限。

### Q2: PhET 模拟实验无法打开？
**A:** PhET 链接会在新窗口打开，请检查浏览器是否阻止了弹窗。

### Q3: 搜索功能不工作？
**A:** 确保已加载课程数据，检查浏览器控制台是否有错误信息。

### Q4: 样式显示异常？
**A:** 确保 Tailwind CSS 已正确配置，运行 `npm install` 重新安装依赖。
