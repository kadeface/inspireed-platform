# 学生管理批量导入模板修改总结

## 🎯 修改目标

## 🎯 用户体验提升

### 修改前的问题

1. ❌ 年级使用名称（七年级、高一）→ 系统需要转换
2. ❌ 班级使用名称（七年级1班）→ 不直观
3. ❌ 学校使用名称 → 可能重复或模糊匹配
4. ❌ 学籍号可选 → 学生无法唯一标识

### 修改后的优势

1. ✅ 年级使用数字（7、10）→ 直接对应系统数据
2. ✅ 班级使用编号（701、1001）→ 唯一且明确
3. ✅ 学校使用代码 → 精确匹配
4. ✅ 学籍号必填 → 确保学生唯一性
5. ✅ 与班级导入格式一致 → 降低学习成本

## 📝 使用指南

### 1. 下载模板

在**组织架构管理 -> 人员管理 -> 学生管理**页面：
1. 点击"📥 批量导入"按钮
2. 点击"📥 下载导入模板"
3. 文件名：`学生导入模板.xlsx`

### 2. 填写数据

**必填字段**（5个）：
- **学号***：学生登录名（如：2024100001）
- **姓名***：学生真实姓名
- **学籍号***：唯一标识（身份证号等）
- **年级级别***：数字（1-12）
- **班级编号***：班级代码（如701、1001）

**可选字段**（4个）：
- 邮箱：自动生成（可留空）
- 手机号：联系方式
- 性别：男/女
- 学校代码：学校编码（可留空，学校管理员模式）

### 3. 上传导入

1. 选择填写好的Excel文件
2. 预览导入数据（显示前几行）
3. 点击"开始导入"
4. 查看导入结果

### 4. 验证结果

导入成功后：
- 检查成功/失败数量
- 查看错误信息（如果有）
- 学生可以登录（默认密码：123456）
- 验证学生关联到正确的班级

## ⚠️ 注意事项

### 数据准备

1. **确保班级已存在**
   - 学生导入前，对应的班级必须已在系统中创建
   - 使用"班级管理"批量导入班级
   - 确认班级编号正确（年级级别 + 班级序号）

2. **学校代码获取**
   - 在"学校管理"中查看学校代码
   - 或使用学校管理员模式（不需要填写学校代码）

3. **学籍号唯一性**
   - 学籍号是学生的唯一标识
   - 不能重复
   - 建议使用身份证号或学号

### 常见错误

| 错误信息 | 原因 | 解决方法 |
|---------|------|---------|
| 学校代码 'XXX' 未找到 | 学校代码不正确 | 检查学校代码，使用正确的编码 |
| 年级级别 X 不存在 | 年级未创建 | 先在系统中创建对应年级 |
| 班级编号 'XXX' 不存在 | 班级未创建 | 先用班级导入创建班级 |
| 学籍号 'XXX' 已存在 | 学生已存在 | 设置"更新已存在学生"或使用不同学籍号 |
| 班级编号格式不正确 | 编号格式错误 | 使用正确格式（如701、1001） |
