## 💡 使用优势

### 为什么同时使用学校名称和学校代码？

1. **学校名称***（必填）
   - 用于后端精确匹配学校
   - 使用学校全名，如"示例学校"、"第一中学"
   - 确保数据准确性

2. **学校代码**（可选）
   - 辅助验证学校信息
   - 数字编码，如"10001"、"20001"
   - 便于快速识别和核对

### 匹配逻辑

后端导入时会：
1. **优先使用"学校名称*"**进行匹配（必填）
2. **如果提供"学校代码"**，会验证代码与名称是否一致
3. **提高数据准确性**：双重验证机制

## ⚠️ 常见错误

### 1. 学校名称不存在

**错误**: `学校 'XXX' 未找到`

**原因**: 填写的学校名称与系统中注册的学校名称不匹配

**解决**:
- 在"学校管理"中查看准确的学校名称
- 确保名称完全匹配（包括空格、标点、全角/半角）
- 或使用"学校批量导入"先创建学校

### 2. 学校代码不匹配

**错误**: `学校代码 'XXX' 与学校名称不匹配`

**原因**: 学校代码与学校名称不一致

**解决**:
- 检查学校管理页面的正确代码
- 或者留空"学校代码"字段（可选字段）
- 只填写"学校名称*"也可以正常导入

### 3. 班级不存在

**错误**: `班级编号 'XXX' 不存在`

**原因**: 学生导入前，对应的班级必须已在系统中创建

**解决**:
- 先在"班级管理"中批量导入班级
- 确认班级编号格式正确（年级级别 + 班级序号）
- 检查班级是否关联到正确的学校

### 4. 学籍号重复

**错误**: `学籍号 'XXX' 已存在`

**原因**: 该学籍号的学生已存在

**解决**:
- 检查是否已导入过该学生
- 或勾选"更新已存在学生"选项（如果支持）
