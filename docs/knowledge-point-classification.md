# 知识点分类规范

## 概述

本文档定义了小学奥数知识点的分类体系和使用规范，用于在资源库中组织和查找知识点相关的互动课件。

## 分类体系

### 一级分类（大类）

1. **计算类** - 涉及数字运算和计算方法
2. **几何类** - 涉及图形、空间和几何概念
3. **应用题类** - 涉及实际问题的数学建模和解决
4. **逻辑推理类** - 涉及逻辑思维和推理能力
5. **数论类** - 涉及数的性质和规律
6. **代数类** - 涉及代数表达式和方程

### 二级分类（小类）

#### 计算类
- 速算技巧
- 巧算方法
- 数论基础

#### 几何类

## 命名规范

### 知识点分类格式

知识点分类使用以下格式存储：
```
大类/小类
```

例如：
- `计算类/速算技巧`
- `几何类/图形认知`
- `应用题类/行程问题`

### 知识点名称格式

知识点名称应该简洁明了，能够准确描述该知识点的核心内容。

示例：
- `乘法口诀可视化`
- `认识基本图形`
- `速算练习`
- `图形面积计算`

### 资源标题格式

建议的资源标题格式：
```
[年级] [知识点名称]
```

例如：
- `三年级 乘法口诀可视化`
- `一年级 图形认知互动`
- `跨年级 速算练习`

## 使用指南

### 上传资源时

1. 选择学科：选择"数学"学科
2. 选择年级：选择对应的年级，或选择"跨年级通用"
3. 选择知识点分类：
   - 先选择大类（如：计算类）
   - 再选择小类（如：速算技巧）
4. 输入知识点名称：输入具体的知识点名称（如：乘法口诀可视化）

### 搜索资源时

1. 在资源库中，可以通过以下方式查找：
   - 按学科 > 年级 > 知识点分类浏览
   - 使用搜索功能，输入知识点名称或分类关键词
   - 在资源卡片上查看知识点分类标签

2. 筛选功能：
   - 点击树形目录中的知识点分类节点
   - 系统会自动筛选出该分类下的所有资源

## 最佳实践

1. **一致性**：使用统一的分类和命名规范
2. **准确性**：确保知识点分类和名称准确反映内容
3. **完整性**：尽量填写完整的分类信息，便于后续查找
4. **描述性**：在资源描述中补充更多信息，如适用年级、核心概念等

## 扩展说明

如需添加新的分类，请：
1. 更新 `frontend/src/config/knowledgePoints.ts` 中的分类配置
2. 更新本文档的分类体系说明
3. 确保新分类符合整体分类逻辑
