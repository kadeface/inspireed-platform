# InspireEd 系统演示文稿文档

本目录包含 InspireEd 系统的演示文稿相关文档，用于40分钟的系统演示。

## 📁 文档说明

### 1. 演示文稿提纲_40分钟.md
**详细的内容提纲**，包含：
- 完整的演示结构
- 每个部分的详细内容
- 时间分配建议
- 演示要点检查清单

**适用场景**：
- 准备演示内容
- 撰写演示脚本
- 理解演示逻辑

### 2. PPT大纲_40分钟.md
**简化的PPT制作大纲**，包含：
- 30张幻灯片的详细清单
- 每张幻灯片的内容要点
- 视觉设计建议
- 演示技巧建议

**适用场景**：
- 制作PPT演示文稿
- 快速了解演示结构
- 准备演示材料

## 🎯 核心观点

> **基于学习科学的日常基础教学行为 + AI赋能 = 教师教学赋能 + 学生学习赋能**

## 📊 演示结构

| 部分 | 时长 | 重点 |
|-----|------|------|
| 开场 | 3分钟 | 教育痛点+系统定位 |
| 第一部分：教育学维度 | 12分钟 | 学习科学理论基础+日常教学行为 |
| 第二部分：教育技术维度 | 18分钟 | 系统框架+AI赋能 |
| 第三部分：学校推广维度 | 5分钟 | 推广策略+价值体现 |
| 总结 | 2分钟 | 核心观点重申 |

## 🎤 使用建议

### 准备演示
1. 先阅读《演示文稿提纲》，理解整体逻辑
2. 根据《PPT大纲》制作幻灯片
3. 准备系统演示环境
4. 准备真实案例和数据

### 演示技巧
- **开场**：用真实场景引入，引起共鸣
- **第一部分**：强调理论如何指导实践
- **第二部分**：现场演示系统功能
- **第三部分**：分享真实案例（如果有）
- **总结**：重申核心观点，激发行动意愿

### 时间控制
- 严格按照时间分配
- 留出5-10分钟互动时间
- 准备Q&A回答
