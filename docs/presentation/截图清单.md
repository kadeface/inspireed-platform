## 📝 截图操作注意事项

1. **浏览器设置**:
   - 建议使用Chrome或Firefox浏览器
   - 截图前确保浏览器窗口足够大，能显示完整内容
   - 可以使用浏览器的全屏模式（F11）来获得更大的截图区域

2. **截图工具**:
   - 可以使用浏览器自带的截图工具
   - 或者使用系统截图工具（Mac: Cmd+Shift+4, Windows: Win+Shift+S）
   - 建议使用全屏截图，然后裁剪需要的部分

3. **截图质量**:
   - 确保截图清晰，文字可读
   - 避免包含敏感信息（如真实姓名、邮箱等）
   - 如果界面中有测试数据，确保数据合理

4. **多设备展示**:
   - 对于"多设备展示"的需求，可以：
     - 在同一台电脑上打开多个浏览器窗口（不同账号）
     - 或者：使用浏览器的响应式设计模式模拟不同设备
     - 或者：分别截图不同设备，然后在PPT中组合展示

5. **实时同步演示**:
   - 对于需要展示"实时同步"的场景：
     - 先准备好教师端和学生端
     - 在教师端进行操作（如切换模块）
     - 快速切换到学生端截图，展示同步效果
     - 或者：使用录屏工具录制操作过程

6. **数据准备**:
   - 某些界面需要先有数据才能显示
   - 建议先创建一些测试教案、课程、活动等
   - 确保有足够的测试数据来展示功能
