# 📍 "开始活动" 按钮在哪里？

## 问题：找不到 "🎯 开始活动" 按钮

## 答案：按钮在导播台的下方

### 🎯 按钮位置示意图

```
┌─────────────────────────────────────────────────────────┐
│  👥 在线学生                                             │
│  [学生列表...]                                           │
└─────────────────────────────────────────────────────────┘

│  ┌────┐    ┌────┐    ┌────┐    ┌────┐    ┌────┐       │

│  │文本│    │视频│    │活动│    │代码│    │文本│       │  ← 这是导播台的模块列表
│  └────┘    └────┘    └──▲─┘    └────┘    └────┘       │
│                         │                                │

```

## ⚠️ 为什么看不到按钮？

### 原因 1：没有选中任何模块 ❌

**表现**：导播台下方完全没有"当前模块详情"区域

**解决**：
1. 在导播台中找到类型为 **"活动"** 的模块
2. **点击**该模块
3. 模块会变成**蓝色**（表示选中）
4. 按钮就会出现

### 原因 2：选中的不是活动类型 ❌

**表现**：导播台下方有"当前模块详情"，但显示的是"文本"、"视频"等其他类型，没有按钮

```
当前模块详情
┌────┐ 文本
│文本│ 第一章介绍
└────┘

(没有按钮，因为这不是活动类型)
```

**解决**：
1. 在导播台中找到显示 **"📋"** 图标或标签为 **"活动"** 的模块
2. 点击那个模块
3. 按钮就会出现

### 原因 3：活动已经启动了 ✅

**表现**：看到的是 **"✅ 结束活动"** 按钮（绿色）而不是 "🎯 开始活动"（蓝色）

```
当前模块详情
┌────┐ 活动  🎯  ← 有这个标记说明活动正在进行
│活动│ 问答练习
└────┘

[✅ 结束活动]  ← 这说明活动已经启动了！
```

**解决**：
- **活动已经启动**！提交面板应该已经在下方显示了
- 如果看不到提交面板，请查看浏览器控制台的错误信息

## 📋 完整的操作步骤（带截图位置）

### 第 1 步：开始上课
```
位置：页面顶部的控制按钮区域
点击：[▶️ 开始上课] 按钮
结果：状态变为 "上课中"（绿色）
```

### 第 2 步：找到导播台
```
位置：在线学生列表的下方
外观：横向排列的模块节点，像链条一样
     ┌────┐    ┌────┐    ┌────┐
     │ 1  │────│ 2  │────│ 3  │
     └────┘    └────┘    └────┘
```

### 第 3 步：识别活动模块
```
特征：
- 图标：📋 (剪贴板图标)
- 颜色：紫色边框
- 标签：下方显示 "活动" 字样
- 类型标记：如果鼠标悬停会显示 "活动" 提示

如何找到：
在导播台的模块中，从左到右查看，找到紫色边框的模块
```

### 第 4 步：点击活动模块
```
操作：点击活动模块的中央区域（不是复选框）
效果：
- 模块背景变成**蓝色**
- 模块有蓝色光晕（高亮效果）
- 导播台下方出现 "当前模块详情" 区域
```

### 第 5 步：找到按钮
```
位置：导播台正下方的 "当前模块详情" 区域
外观：
  当前模块详情
  ┌────┐ 活动
  │活动│ [活动名称]
  └────┘
  
  [🎯 开始活动]  ← 蓝色按钮，在这里！
```

### 第 6 步：点击开始活动
```
操作：点击 [🎯 开始活动] 按钮
效果：
- 按钮变成 [✅ 结束活动]
- 活动模块右上角出现 🎯 标记
- 下方出现 "📝 活动提交" 面板
```

## 🔍 调试：如何确认自己在正确位置

### 打开浏览器控制台（按 F12）

然后执行以下检查：

#### 检查 1：确认导播台存在
```
如果看到 "📺 导播台" 标题 → ✅ 导播台已加载
如果没有看到 → ❌ 可能会话未创建，先点击"开始上课"
```

#### 检查 2：确认有活动模块
```
在导播台中数一下有几个模块
看看有没有显示 "活动" 或 "📋" 的模块
如果没有 → ❌ 这个教案可能没有活动类型的 Cell
```

#### 检查 3：确认模块被选中
```
被选中的模块应该：
- 背景是蓝色
- 有蓝色光晕
- 比其他模块略大

如果没有模块是蓝色的 → ❌ 请点击活动模块
```

## ❓ 还是找不到？

### 可能性 1：教案中没有活动

**检查方法**：
1. 进入教案编辑器
2. 查看左侧的模块列表
3. 看有没有类型为 "Activity" 的模块

**如果没有**：
- 需要先在教案编辑器中添加活动模块
- 保存教案
- 刷新课堂页面

### 可能性 2：页面布局问题

**尝试**：
1. 按 F11 退出全屏模式
2. 调整浏览器窗口大小
3. 向下滚动页面（导播台可能在视口外）
4. 刷新页面（Ctrl+R 或 Cmd+R）

### 可能性 3：权限或数据问题

**检查控制台**：
1. 按 F12 打开开发者工具
2. 切换到 Console 标签
3. 看有没有红色错误信息
4. 截图发给技术支持

## 💡 小贴士

- 导播台的模块可以**水平滚动**，如果模块很多，活动可能在右侧看不到，可以向右滚动查看
- 如果导播台显示的是"预览模式"，说明还没有开始上课
- "当前模块详情"只有在**选中模块后**才会出现
- 按钮的颜色：🎯开始活动(蓝色) vs ✅结束活动(灰绿色)
