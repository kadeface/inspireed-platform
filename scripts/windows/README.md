# Windows 脚本目录

本目录包含 Windows 系统的批处理脚本（.bat 文件）。

## 文件说明

### `start.bat`
Windows 环境下启动 InspireEd 服务的脚本。

### `stop.bat`
Windows 环境下停止 InspireEd 服务的脚本。

### `restart.bat`
Windows 环境下重启 InspireEd 服务的脚本。

## 使用方法

如果需要使用这些脚本，可以从此目录复制到项目根目录，或者直接在此目录运行：

```cmd
cd scripts\windows
start.bat
```

## 注意

- 这些脚本是为 Windows 系统准备的
- 在 macOS/Linux 系统上，请使用对应的 `.sh` 脚本
- 如果需要在 Windows 上使用，建议将脚本链接或复制到项目根目录
