# 维护脚本

本目录存放用于诊断、检查和修复的 Python 脚本。

| 脚本 | 说明 |
|------|------|
| add_teacher_to_classroom.py | 添加教师到班级 |
| analyze_user_relationships.py | 分析用户关联关系 |
| check_users_classroom.py | 检查用户班级关系 |
| check_users_simple.py | 简单用户检查 |
| diagnose_classrooms.py | 班级诊断 |
| diagnose_student_join.py | 学生加入课堂诊断 |
| quick_fix_student.py | 学生问题快速修复 |
| test_lesson_list_performance.py | 教案列表性能测试 |
| test_performance.sh | 性能测试脚本 |

运行前需在项目根目录执行，或设置正确的 PYTHONPATH。
