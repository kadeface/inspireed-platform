## 💡 使用技巧

1. **学习进度自动保存**：无需担心数据丢失
2. **笔记1秒自动保存**：专注学习，系统自动保存
3. **代码即时执行**：Python代码在浏览器中运行
4. **成就自动解锁**：完成任务即可获得徽章

## ⚠️ 重要提示

- 学习数据存储在浏览器localStorage中
- 清除浏览器数据会丢失学习记录
- Python首次执行需加载环境（约3-5秒）
