## 🐛 如果遇到问题

### 1. 依赖安装失败
```bash
# 清理缓存
cd frontend
rm -rf node_modules pnpm-lock.yaml
pnpm install
pnpm add uuid vuedraggable idb
```

### 2. 页面报错
- 强制刷新页面
- 检查浏览器控制台
- 查看后端是否运行

### 3. 离线功能不工作
- 确认已安装 `idb` 包
- 检查浏览器是否支持 IndexedDB
- 查看浏览器控制台错误

### 4. 互评功能问题
- 确认至少有 3 个学生提交
- 检查后端 API 是否正常
- 查看网络请求状态
