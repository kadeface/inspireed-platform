# 学科教研组功能 - 快速开始指南

## 一分钟了解

学科教研组功能让教师可以：
- 📚 创建和加入学科教研组
- 👥 管理教研组成员
- 🎓 分享和查看优质教学设计
- 📊 查看教研组统计数据

## 快速启动

### 1. 确保系统已启动

```bash
# 在项目根目录执行
./start.sh
```

或手动启动：

```bash
# 启动后端
cd backend
source venv/bin/activate
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 启动前端（新终端窗口）
cd frontend
npm run dev
```

### 2. 数据库迁移

如果是首次使用，需要执行数据库迁移：

```bash
cd backend
source venv/bin/activate
alembic upgrade head
```

### 3. 访问功能

1. 打开浏览器访问：`http://localhost:5173/login`
2. 使用教师账号登录
3. 在导航栏或菜单中找到"学科教研组"入口
4. 或直接访问：`http://localhost:5173/teacher/subject-groups`

## 快速使用教程

### 场景一：创建教研组

1. **进入教研组页面**
   - 访问 `/teacher/subject-groups`
   
2. **点击"创建教研组"按钮**
   
3. **填写信息**
   - 教研组名称：例如"智能农业教研组"
   - 所属学科：选择"智能农业"
   - 教研组范围：选择"校级/区域级/全国级"
   - 教研组描述：介绍教研组的目标
   - 是否公开：勾选后其他教师可以看到
   
4. **提交创建**
   - 创建成功后，你将自动成为组长

### 场景二：邀请成员

1. **进入教研组详情页**
   - 在教研组列表中点击某个教研组卡片
   
2. **切换到"成员管理"标签**
   
3. **点击"添加成员"按钮**
   
4. **输入用户信息**
   - 用户ID：要添加的教师ID
   - 成员角色：选择"普通成员"或"管理员"
   
5. **提交添加**
   - 成功后成员可以访问教研组

> 💡 提示：如何获取用户ID？
> - 联系管理员获取
> - 或在用户管理页面查看

### 场景三：分享教学设计

1. **进入教研组详情页**
   
2. **切换到"共享教学设计"标签**
   
3. **点击"分享教案"按钮**
   
4. **选择要分享的教学设计**
   - 下拉列表显示你创建的所有教学设计
   - 选择一个你想分享的
   
5. **添加分享说明**（可选）
   - 介绍这个教学设计的特点
   - 提供使用建议
   
6. **提交分享**
   - 分享成功后，组内所有成员都能看到

### 场景四：查看和使用共享的教学设计

1. **进入教研组详情页**
   
2. **浏览"共享教学设计"列表**
   
3. **点击"查看"按钮**
   - 查看教学设计的详细内容
   - 系统会自动记录查看次数
   
4. **点击"复制"按钮**
   - 将优秀的教学设计复制到自己的账号
   - 可以在此基础上进行修改
   - 系统会自动记录下载次数

## 角色权限说明

### 组长 (Owner)
- ✅ 查看教研组信息
- ✅ 编辑教研组信息
- ✅ 删除教研组
- ✅ 添加/移除/编辑所有成员
- ✅ 分享自己的教学设计
- ✅ 移除任何人分享的教学设计

### 管理员 (Admin)
- ✅ 查看教研组信息
- ✅ 编辑教研组信息
- ✅ 添加/移除/编辑普通成员
- ✅ 分享自己的教学设计
- ✅ 移除任何人分享的教学设计

### 普通成员 (Member)
- ✅ 查看教研组信息
- ✅ 分享自己的教学设计
- ✅ 查看所有共享的教学设计
- ✅ 复制共享的教学设计
- ✅ 移除自己分享的教学设计
- ✅ 退出教研组

## 常见问题

### Q1: 我能同时加入多个教研组吗？
**A:** 可以！一个教师可以加入多个教研组，没有数量限制。

### Q2: 如何找到其他教研组？
**A:** 在教研组列表页面：
- 使用筛选功能按学科、范围筛选
- 勾选"仅显示我加入的教研组"查看自己的教研组
- 查看公开的教研组并申请加入（功能待开发）

### Q3: 能分享别人的教学设计吗？
**A:** 不能。只能分享自己创建的教学设计。这确保了分享者对内容负责。

### Q4: 复制的教学设计会同步更新吗？
**A:** 不会。复制后的教学设计是独立的，原作者的修改不会影响你的副本。

### Q5: 如何管理教研组成员？
**A:** 组长和管理员可以：
- 添加新成员
- 修改成员角色（普通成员 ↔ 管理员）
- 移除成员（不能移除组长）

### Q6: 教研组可以删除吗？
**A:** 可以，但只有组长可以删除。删除是软删除，数据库中保留记录。

### Q7: 统计数据多久更新一次？
**A:** 实时更新。每次操作后统计数据会立即刷新。

### Q8: 能给共享的教学设计评论吗？
**A:** 目前还不支持，这是未来的优化方向。

## 数据统计说明

在教研组列表页顶部，你会看到5个统计卡片：

1. **全部教研组** - 系统中所有激活的教研组数量
2. **我的教研组** - 你作为成员加入的教研组数量
3. **总成员数** - 所有教研组的成员总数
4. **共享教学设计** - 所有教研组共享的教学设计总数
5. **我的分享** - 你分享的教学设计总数

## 教研组类型说明

### 校级教研组
- 适合：同一学校的教师
- 范围：限定在一个学校
- 例如：某某小学智能农业教研组

### 区域级教研组
- 适合：同一区域的教师
- 范围：限定在一个区域（省/市/区）
- 例如：北京市智能农业教研组

### 全国级教研组
- 适合：所有教师
- 范围：不限制地域
- 例如：全国智能农业教学研究组

## 最佳实践

### 创建教研组
1. **明确主题** - 给教研组一个清晰的名称和描述
2. **合理分类** - 选择正确的学科和范围
3. **设置公开** - 如果希望更多教师加入，建议设置为公开

### 管理成员
1. **角色分配** - 根据成员的参与度分配角色
2. **定期审查** - 定期检查成员列表，移除不活跃成员
3. **授权管理员** - 选择活跃的成员作为管理员协助管理

### 分享教学设计
1. **添加说明** - 详细说明教学设计的特点和适用场景
2. **保持更新** - 如果有更好的版本，重新分享
3. **质量优先** - 分享经过验证的优质教学设计

### 使用共享资源
1. **先查看再复制** - 确认教学设计符合需求再复制
2. **适当修改** - 根据自己的教学环境适当调整
3. **反馈改进** - 向分享者提供反馈（通过其他渠道）
