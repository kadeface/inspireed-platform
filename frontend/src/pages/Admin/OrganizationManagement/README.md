# OrganizationManagement 组件重构说明

## 重构概述

将组织架构管理模块从原来的5个独立卡片重构为**四层架构**，更加清晰、符合教育机构的组织结构。

## 新架构设计

```
OrganizationManagement/
├── RegionManagementCard.vue          # 1. 区域管理
├── SchoolManagementCard.vue          # 2. 学校管理
├── ClassroomManagementCard.vue       # 3. 班级管理（已简化）
├── PersonnelManagementCard.vue       # 4. 人员管理（新增）
│   ├── TeacherManagementTab.vue      #    教师管理标签页
│   │   ├── TeacherProfileTab.vue     #      - 教师档案
│   │   ├── TeacherAssignmentTab.vue  #      - 教学任务（迁移自原组件）
│   │   └── PositionTypeTab.vue       #      - 职务类型配置（迁移自原组件）
│   └── StudentManagementTab.vue      #    学生管理标签页
│       └── StudentProfileTab.vue     #      - 学生档案
└── README.md                          # 本文档

## 保留的原组件（已迁移）
├── TeacherAssignmentCard.vue         # 已迁移到 TeacherAssignmentTab.vue
└── PositionTypeCard.vue              # 已迁移到 PositionTypeTab.vue
```

## 主要变更

### 1. 新增人员管理模块

将原有的**教师教学任务**和**职务类型管理**整合到新的**人员管理**模块中，并新增**教师档案**和**学生管理**功能。

**人员管理结构：**
```
人员管理
├── 教师管理
│   ├── 教师档案（基本信息、所属学校、职务）
│   ├── 教学任务（原教师教学任务功能）
│   └── 职务类型配置（原职务类型管理功能）
└── 学生管理
    └── 学生档案（基本信息、所属班级、学籍号）
```

### 2. 简化班级管理

移除班级成员管理功能（已迁移到人员管理的教师/学生档案中），班级管理模块现在只负责：
- 班级CRUD操作
- 批量导入班级
- 班级筛选和查询

### 3. 主入口调整

将原来的5个功能卡片调整为4个：

| 原结构 | 新结构 |
|--------|--------|
| 区域管理 | 区域管理 ✅ |
| 学校管理 | 学校管理 ✅ |
| 班级成员管理 | 班级管理 ✅（已简化） |
| 教师教学任务 | 人员管理 ✅（整合） |
| 职务类型管理 | 人员管理 ✅（整合） |

## 组件功能说明

### RegionManagementCard.vue
- 区域CRUD操作
- 区域层级管理
- 区域筛选和搜索

### SchoolManagementCard.vue
- 学校CRUD操作
- 批量导入学校
- 县区端批量导入班级
- 学校筛选和搜索

### ClassroomManagementCard.vue
- 班级CRUD操作
- 批量导入班级
- 班级筛选（区域、学校、年级、搜索）
- **已移除**：班级成员管理功能

### PersonnelManagementCard.vue（新增）
人员管理主入口，包含教师和学生的统一管理。

#### TeacherManagementTab.vue
教师管理标签页，包含三个子标签：

1. **TeacherProfileTab.vue（教师档案）**
   - 教师基本信息管理
   - 所属学校分配
   - 教师状态管理
   - 批量导入教师
   - 重置密码

2. **TeacherAssignmentTab.vue（教学任务）**
   - 教学任务分配（班主任/学科教师）
   - 教学任务筛选（区域、学期、年级、学科、学校、教师）
   - 批量导入教学任务
   - 导出新创建教师账号

3. **PositionTypeTab.vue（职务类型配置）**
   - 自定义职务类型管理（如校长、教研室主任等）
   - 系统职务类型（班主任、学科教师）管理
   - 职务分类（教学类/管理类/行政类）

#### StudentManagementTab.vue
学生管理标签页，包含学生档案管理：

1. **学生档案管理**
   - 学生基本信息（姓名、学号、学籍号）
   - 所属班级分配
   - 学生类型（文科/理科/未分科）
   - 批量导入学生
   - 重置密码

## 优势

1. **逻辑清晰**：按组织层级划分（区域→学校→班级→人员），符合教育机构的实际结构
2. **职责单一**：每个组件职责明确，便于维护
3. **人员集中**：教师和学生的管理集中在一个模块，减少页面跳转
4. **便于扩展**：未来可以轻松添加更多人员类型或管理功能

## 迁移说明

### 已迁移的组件
- `TeacherAssignmentCard.vue` → `TeacherAssignmentTab.vue`
- `PositionTypeCard.vue` → `PositionTypeTab.vue`

原组件文件保留，可以继续使用或删除。

### 数据流
- 所有人员相关的筛选、导入、导出功能保持不变
- API调用保持兼容
- 用户数据模型保持一致
