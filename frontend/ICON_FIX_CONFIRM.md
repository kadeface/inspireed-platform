## 🐛 问题原因

Element Plus图标组件的使用方式不正确。在Vue 3中，当图标已经全局注册时，需要使用动态组件的方式来引用。

### 修复前（错误）

```vue
<el-icon :size="40">
  <Calendar />
</el-icon>
```

### 修复后（正确）

```vue
<el-icon :size="40">
  <component :is="'Calendar'" />
</el-icon>
```

## 💡 技术细节

### 为什么使用动态组件？

在Vue 3中，当图标通过以下方式全局注册时：

```javascript
// main.ts
for (const [key, component] of Object.entries(ElementPlusIconsVue)) {
  app.component(key, component)
}
```

图标会被注册为全局组件，但是在模板中直接使用 `<Calendar />` 可能无法正确解析。

**解决方案**: 使用Vue的动态组件语法

```vue
<component :is="'Calendar'" />
```

这样可以确保Vue能够正确查找并渲染已注册的全局组件。
