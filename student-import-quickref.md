## 🎯 核心变化

### 修改前 → 修改后

|  | 修改前 | 修改后 |
|--|--------|--------|
| 年级 | 年级*（七年级、高一） | 年级级别*（7、10） |
| 班级 | 班级*（七年级1班） | 班级编号*（701、1001） |
| 学校 | 所属学校*（学校名称） | 学校代码（10001） |
| 学籍号 | 可选 | 必填* |
| 邮箱 | 必填* | 可选 |
| 新增 | - | 手机号 |
| 新增 | - | 性别 |

### 1. 下载模板

```
组织架构管理 → 人员管理 → 学生管理 → 📥 批量导入 → 📥 下载导入模板
```

### 2. 填写数据（最小必填）

| 学号* | 姓名* | 学籍号* | 年级级别* | 班级编号* |
|-------|-------|---------|----------|----------|
| 2024100001 | 张三 | 123456789012345678 | 7 | 701 |
| 2024100002 | 李四 | 987654321098765432 | 7 | 702 |

### 3. 上传导入

1. 选择Excel文件
2. 预览数据
3. 点击"开始导入"

### 4. 验证

- 学生可以登录（默认密码：123456）
- 学生关联到正确的班级

## ⚠️ 常见错误

### 1. 班级编号格式错误

❌ 错误：`七年级1班`
✅ 正确：`701`

### 2. 年级使用名称

❌ 错误：`七年级`
✅ 正确：`7`

### 3. 学籍号为空

❌ 错误：不填学籍号
✅ 正确：填写唯一标识（如身份证号）

### 4. 班级不存在

❌ 错误：导入学生时班级未创建
✅ 正确：先在"班级管理"中导入班级

## 💡 提示

### 数据准备

1. **学校代码获取**：在"学校管理"页面查看
2. **班级编号规则**：年级（1-12）+ 班级序号（01-99）
3. **学籍号建议**：使用身份证号或统一学号

### 批量导入建议

1. 每次不超过500条
2. 分学校、分年级导入
3. 先导入班级，再导入学生
4. 保留原始Excel备份
